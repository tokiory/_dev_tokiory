# Общий обзор
**Zig** —  это язык общего назначения со встроенным тулкитом, который позволяет разрабатывать гибкое и стрессоустойчивое ПО.

Он позволяет явно управлять памятью, работать с указателями и байтовыми данными.

Давайте рассмотрим простой сниппет:

```zig
const std = @import("std");

pub fn main() void {
  const hello = "hello"
  const world = "world"
  std.debug.print("{} {}\n", .{ foo, bar });
}
```

Данный сниппет представляет из себя простой кусок кода, который выводит нам два числа, разберем данный код по частям.

```zig
const std = @import("std"); // [!code focus]

pub fn main() void {
  var foo: i8 = 50;
  const bar: u32 = 314159;
  std.debug.print("{} {}\n", .{ foo, bar });
}
```

Сначала мы импортируем стандартную библиотеку `std`, которая содержит в себе множество полезных функций и структур данных.
С помощью неё мы в будущем будем выводить в стандартный поток нужные нам данные.

```zig
const std = @import("std");

// [!code focus:5]
pub fn main() void {
  var foo: i8 = 50;
  const bar: u32 = 314159;
  std.debug.print("{} {}\n", .{ foo, bar });
}
```

Затем мы объявляем функцию `main`, которая будет являться стартовой точкой для выполнения нашей программы.

По умолчанию все функции в Zig —  приватные, поэтому нам нужно объявить данную функцию с модификатором `pub`
</CodeSlide>

<CodeSlide>

```zig
const std = @import("std");

pub fn main() void {
  // [!code focus:2]
  var foo: i8 = 50;
  const bar: u32 = 314159;
  std.debug.print("{} {}\n", .{ foo, bar });
}
```

Здесь мы объявляем новые числовые переменные с типом `i8` и `u32`.

Ключевое слово `var` используется в случае, если переменная может изменить свое значение в будущем, а `const`, наоборот, указывает на то, что переменная не сможет поменять свое значение.
</CodeSlide>

<CodeSlide>

```zig
const std = @import("std");

pub fn main() void {
  var foo: i8 = 50;
  const bar: u32 = 314159;
  // [!code focus]
  std.debug.print("{} {}\n", .{ foo, bar });
}
```

В данном кусочке сниппета мы выводим две наши переменные. Пока что нас не должен беспокоить синтаксис с фигурными скобками, мы до него еще не добрались. Все что стоит знать, так это то, что метод `std.debug.print` выводит значения наших переменных в консоль.
